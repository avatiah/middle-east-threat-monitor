# Geopolitical Threat Monitor (Middle East)

Визуальный дашборд для отслеживания военных рисков на Ближнем Востоке. Данные подтягиваются напрямую из рынков предсказаний Polymarket.

### Как это работает:
1. **Цены акций = Вероятность:** Если акция "Да" стоит 0.30$, система отображает риск 30%.
2. **Администрирование:** Чтобы добавить или убрать угрозу, отредактируйте файл `src/config/threats.json`.

### Как запустить за 2 минуты:
1. Создайте репозиторий на GitHub.
2. Скопируйте файлы проекта.
3. Зайдите на [Vercel](https://vercel.com), импортируйте этот репозиторий.
4. Проект готов!

### Управление угрозами:
В файле `threats.json` измените параметр `"active": false` для неактуальных событий, и они исчезнут с сайта.
